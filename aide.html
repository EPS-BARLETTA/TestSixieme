Mode d’emploi

1) Démarrer

Sur la page d’accueil, clique sur Démarrer puis saisis Nom, Prénom, Classe, Sexe pour les 2 élèves.

2) Choisir un test

Depuis le Menu, lance les tests dans l’ordre de ton choix : Endurance (Luc‑Léger), Saut sans élan, Vitesse 30 m.

3) Saisir les résultats

Chaque page de test enregistre pour les deux élèves avec le bouton Enregistrer & Menu.

4) Générer le QR

Quand les 3 tests sont saisis pour les 2 élèves, clique sur Générer le QR. Un code JSON compatible ScanProf s’affiche (une seule image, 2 élèves).

5) Mode Prof Luc‑Léger

Réservé aux enseignants (code 57). Affichage centré, bips, barre d’avancement épaisse.
